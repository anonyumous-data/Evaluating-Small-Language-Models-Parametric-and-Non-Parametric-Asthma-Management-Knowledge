All data from the Evaluating-Small-Language-Models-Parametric-and-Non-Parametric-Asthma-Management-Knowledge work.
 * Folder Models_answers includes all Small language models and Large Language Models answers collected in this work
 * Folder gpu_logs includes all GPU data collected during Small Language Model inference
 * Folder rag_storage stores the knowledge graph schema and chunks, ready to use in LightRAG within any machine
 * Folder berts_rouges_blues_scores includes all classical metrics results from responses of all models evaluated in this work
 * GINA_FAQ.json includes questions and answers used in this work from the original GINA FAQ website https://ginasthma.org/about-us/faqs/?_x_tr_sl&_x_tr_tl&_x_tr_hl
   * Note: the context string "the next question is about asthma, {question}" as added in script execution level
* File evaluation_prompt contains de prompt used in the LLM-judge
* File lighrag_general_prompts contains all LightRAG prompts used in the graph creation and retrieval usage
